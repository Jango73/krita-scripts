## [1.0.0] - 2026-03-16

First public release of the ComfyUI Image Enhance plugin for Krita.

This release introduces the core workflow integration between Krita and ComfyUI: send the current image or selected regions to a ComfyUI server, then insert the generated results back into the document as new layers. It includes the main enhancement dialog, a dedicated workflow docker, configuration and prompt management, and support for region-based processing with soft layer integration.
